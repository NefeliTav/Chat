# Chat
The program simulates a simple chat application, using shared memory and semaphores.

## Clean
```
make clean
```

##Compile
```
make
```
## Run - in the first shell
```
./p -p x
```
x stands for the probability of the message changing (e.g. 0.2)

## Run - in the second shell
```
./p
```
